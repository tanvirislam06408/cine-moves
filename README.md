# Cine-Moves

![App Screenshot]()

A personal movie watchlist manager built with React. Browse pre-populated movies, add new ones, toggle watched/unwatched status, search, filter, and delete movies with confirmation dialogs.

## Features

- Browse a grid of popular movies with poster images
- Add custom movies via a modal form
- Toggle watched/unwatched status per movie
- Delete movies with a SweetAlert2 confirmation dialog
- Filter movies by All / Watched / Unwatched
- Live search by movie title
- Statistics dashboard (Watched, Unwatched, Total counts)

## Tech Stack

- **React 19** with functional components and hooks
- **Vite 8** for development server and builds
- **Tailwind CSS 4** + **daisyUI 5** for styling
- **react-icons** for icons
- **SweetAlert2** for confirmation dialogs
- Deployed on **Vercel**

## Setup

```bash
# Clone the repository
git clone https://github.com/your-username/cine-moves.git
cd cine-moves

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
